# simia_tui_light

A minimal TUI for fast, timestamped experiment logging with customizable macro hotkeys.

Commands:                             
                                                                                    1 = START RECORDING                 
                                                                                    2 = STOP RECORDING                  
                                                                                    3 = START TASK                      
                                                                                    4 = STOP TASK                       
                                                                                    5 = FIX CAMERA                      
                                                                                    6 = PAUSE TASK                      
                                                                                    7 = RESUME TASK                     
                                                                                    8 = ADJUST LIGHTING/CAMERA          
                                                                                    9 = SLEEPY                          
                                                                                    n = note, m = mark, u = undo, r = reload config, h = help, q = stop
